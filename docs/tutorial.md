# Tutorial: Создание HTTP-сервера с нуля на Python

## 1. Введение

В рамках проекта был разработан простой HTTP-сервер с нуля на языке Python.  
Основной целью проекта являлось изучение принципов работы HTTP-протокола, сокетов и обработки клиент-серверных запросов.

Сервер реализован без использования готовых веб-фреймворков.

---

## 2. Что такое HTTP-сервер

HTTP-сервер — это программное обеспечение, принимающее HTTP-запросы от клиента (браузера) и возвращающее HTTP-ответ.

Основные функции HTTP-сервера:

- приём соединений;
- обработка запросов;
- отправка ответов;
- работа со статическими файлами.

## 4. Создание серверного сокета

Для реализации сервера использовался модуль socket.

Пример создания сокета:

```python
server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_socket.bind((HOST, PORT))
server_socket.listen(5)
```

Описание:

- AF_INET — использование IPv4;
- SOCK_STREAM — TCP-соединение;
- bind() — привязка сервера к адресу и порту;
- listen() — перевод сокета в режим ожидания подключений.

---

## 5. Обработка HTTP-запросов

После подключения клиента сервер принимает запрос:

```python
request = client_socket.recv(1024).decode('utf-8')
```

Из запроса извлекаются:

- HTTP-метод;
- путь к странице;
- версия протокола.

Пример:

```python
request_line = request.split('\n')[0]
method, path, _ = request_line.split()
```

---

## 6. Формирование HTTP-ответа

Сервер формирует HTTP-ответ вручную.

Пример:

```python
response = (
    f'HTTP/1.1 {status_code} OK\n'
    f'Content-Type: text/html\n\n'
    f'{body}'
)
```

Ответ включает:

- HTTP-статус;
- заголовки;
- содержимое страницы.

## 7. Работа со статическими файлами

Сервер умеет отдавать HTML и CSS файлы из папки static.

Получение содержимого файла:

```python
with open(full_path, 'r', encoding='utf-8') as file:
    return file.read(), 200
```

## 8. Реализация ошибки 404

Если файл не найден, сервер возвращает страницу ошибки:

```python
return """
<h1>404 Not Found</h1>
<p>Страница не найдена</p>
""", 404
```

Таким образом реализована обработка несуществующих маршрутов.
