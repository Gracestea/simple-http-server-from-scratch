# simple-http-server-from-scratch
Дополнительное задание для проектной практики 2026

## Описание

Проект представляет собой простой HTTP-сервер,
реализованный с нуля на Python с использованием сокетов.

## Возможности

- обработка HTTP GET запросов
- отдача HTML страниц
- поддержка CSS
- обработка ошибки 404
- логирование запросов
- работа со статическими файлами

## Запуск

```bash
python server.py
